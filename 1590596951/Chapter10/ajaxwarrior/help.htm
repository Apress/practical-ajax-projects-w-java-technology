AJAXWarrior Help
 
Arrow keys   Move your player
A   Toggle between Talk and Attack mode (Blue border for Talk mode, Red for Attack mode)
C   Cast spell
E   Enter community (Must be standing on it)
H   This help window
I   Display inventory
P   Pick up item (Must be standing on it)
S   Save game
W   Switch weapon

-- Press SPACEBAR to return to game --
